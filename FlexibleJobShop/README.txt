Pour exécuter le jar (Java executable), veuillez aller sur votre terminal, déplacez vous dans le dossier contenant le fichier .jar, et taper:
	java -jar jobshop.jar

Pour le numéro d’itérations, vous pouvez simplement taper un entier >0

Pour le ”SOURCE” du fichier que vous souhaitez utiliser, il faut que le chemin soit spécifié en entier.

Un exemple: /Users/gilbertgullberg/Documents/FlexibleJobShop/FlexibleJobShopFinal/Flexible_Job_Shop/Mk02.fjs
